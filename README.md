# Northwind SQL Analysis

## Project Overview

This project contains SQL analysis performed on the Northwind database using PostgreSQL.

## Tools Used

* PostgreSQL
* SQL
* GitHub

## SQL Concepts Demonstrated

* INNER JOIN
* LEFT JOIN
* GROUP BY
* HAVING
* COUNT()
* SUM()
* AVG()
* ORDER BY
* LIMIT

## Business Questions Solved

1. Top 5 customers by order count
2. Top 5 customers by revenue
3. Customers with more than 20 orders
4. Categories with average price greater than 40
5. Products never ordered
6. Employees handling more than 50 orders
7. Top suppliers by product count

## Dataset

Northwind Sample Database
